# NexDroid Control Center

NexDroid Control Center is a premium desktop Android device management companion built with Python, CustomTkinter, SQLite, background workers, and ADB.

## Run

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python main.py
```

ADB must be installed and available on your `PATH`, or configured later in the app settings.
